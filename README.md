# gripper-cobot-esp32
Este repositorio contiene el desarrollo de un gripper robótico integrado a un cobot Elite Robot. Usa un actuador lineal de 12 V, controlado por un ESP32-C3 SuperMini, con indicador NeoPixel. El cuerpo fue impreso en PETG con fibra de carbono y los dedos tipo Fin Ray en TPU 85A para mejorar la sujeción.
